# grigarr/homebrew-tap

Homebrew tap for my personal tools.

## Usage

```sh
brew install grigarr/tap/awake
```

This taps `grigarr/homebrew-tap` automatically. To tap explicitly:

```sh
brew tap grigarr/tap
brew install awake
```

## Formulae

| Formula | Description |
|---------|-------------|
| [`awake`](Formula/awake.rb) | Cleanly see (and act on) what is keeping your Mac awake — [repo](https://github.com/grigarr/awake) |
